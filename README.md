# 🎓 STUDENT RECORD MANAGEMENT SYSTEM

## 📖 Description

The **Student Record Management System** is a Python-based application.

It is designed to manage student records efficiently.

The project uses **Binary File Handling (Pickle)** for permanent data storage.

Student information is stored securely in a binary **`.dat`** file.

The system supports adding, displaying, searching, updating, and deleting student records.

It automatically calculates the **total marks** and **percentage** of each student.

The application demonstrates the practical use of Python programming concepts such as:

* 🐍 Object-Oriented Programming (OOP)
* 📂 Binary File Handling (Pickle)
* 🔍 Binary Search Algorithm
* 🔄 Functions & Modular Programming
* 💾 Data Storage & Validation

The project provides a simple and user-friendly **Command Line Interface (CLI)** for easy interaction.

---

## ✨ Key Features

* ➕ Add new student records
* 🚫 Prevent duplicate roll number entries
* 🏫 Store school/college information
* 📚 Store subject-wise marks
* 🧮 Automatically calculate total marks
* 📊 Automatically calculate percentage
* 🔍 Search student records using Binary Search
* ✏️ Update existing student details
* 🗑️ Delete student records
* 💾 Store data permanently using **`students.dat`**
* 🖥️ Menu-driven Command Line Interface (CLI)
