# STUDENT-RECORD-SYSTEM
A menu-driven Student Record Management System built using Python and Binary File Handling (pickle). The project allows users to add, display, search, update, and delete student records while storing data permanently in a binary .dat file.

Features
* Add student records with duplicate roll number validation
* Store school/college information and subject-wise marks
* Automatic total marks and percentage calculation
* Search records using Binary Search
* Update existing student details
* Delete specific student records
* Persistent data storage using binary files (students.dat)
* User-friendly command-line interface
Technologies Used
* Python
* Object-Oriented Programming (OOP)
* File Handling (pickle)
* Binary Search Algorithm
